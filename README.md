# WechatLuckyMoney

微信抢红包插件，基于 Xposed 框架，速度快得冷人发指。为避免过度拉仇恨，请谨慎使用。


## 包含黑名单不抢

免打扰功能每个微信版本升级都需要适配，现在换了思路，可以设置白名单。需要先打开显示微信ID 的功能，点进微信个人信息页或者群信息页，能查看并自动复制微信 ID，把他粘贴到"包含微信ID不抢"的设置项里面去，多个ID 以逗号分隔。
